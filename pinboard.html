Pinboard.css

Pinboard.css is an adaptation of—and a userstyle for—Pinboard. Though not a significant departure from the default stylesheet, it is an attempt to modify the design and layout to bring about a touch more clarity and legibility.

At the same time the layout is reworked to be responsive. This works across most pages on Pinboard and across most screen sizes. There are some exceptions, though, which exist because of either the HTML document's structure or inline styles. When possible I've worked around—or with—these limitations.

To try Pinboard.css you will need to install the extension Stylus (Firefox, Chrome, Opera) and import the stylesheet. It may also work with Stylish, but I haven't tested it at this point in time.
